Desglose de los Componentes

💻

Claude Code (El Arnés Local)

La herramienta de línea de comandos (CLI) que instalas. Actúa como el intermediario: gestiona las sesiones, envía peticiones a la API y ejecuta físicamente las herramientas en tu máquina (lee archivos, corre comandos).

🧠

El Modelo Claude

La inteligencia en la nube. No tiene acceso directo a tus archivos. En su lugar, lee un "menú" de herramientas que el CLI le envía y razona: "Para resolver esto, primero necesito pedirle al CLI que ejecute un 'ls' en esta carpeta".

🛠️

Las Herramientas (Tools)

Lo que transforma a un chatbot en un agente. Las herramientas permiten a Claude interactuar con la realidad: buscar en el código, editar líneas específicas, ejecutar tests (bash), leer logs de errores o hacer commits en Git.

📄

CLAUDE.md

Tu archivo de memoria y reglas. El CLI no lo "entiende", simplemente lo adjunta como texto en cada petición para que el modelo sepa cuáles son los estándares de tu equipo, comandos de build o arquitectura preferida.

🔄 Las 3 Fases del Bucle Agéntico

A diferencia de una simple respuesta de chat, Claude Code opera en un ciclo continuo hasta resolver el problema:

  • 1. Recopilar Contexto (Explore): Ante una petición (ej. "Arregla el bug de login"), Claude primero pide ejecutar búsquedas en los archivos para entender cómo funciona la autenticación en tu proyecto específico.
  • 2. Tomar Acción (Plan & Implement): Con la información recopilada, Claude decide qué código cambiar y envía una petición estructurada para que la herramienta local edite los archivos exactos.
  • 3. Verificar Resultados (Verify): Claude Code se distingue por su capacidad de autoevaluación. Puede pedir ejecutar los tests de tu proyecto. Si fallan, lee el error, vuelve a razonar y corrige su propio código.

📞 La Analogía del Asesor Remoto

Imagina cómo funcionan juntos el modelo y la herramienta local:

  • 👤 Tú: Le pides a tu asistente local que resuelva un problema complejo.
  • 💻 Asistente Local (CLI): Está en tu oficina. Tiene manos para abrir archivos y teclear, pero necesita que le digan qué hacer. Llama por teléfono a un Experto.
  • 🧠 Experto Remoto (Claude): Es un genio, pero no tiene acceso físico a tu oficina. Le dice al Asistente: "Abre la carpeta de cuentas y dítame qué archivos hay".
  • 🔄 El Bucle: El Asistente lo hace y se lo lee por teléfono. El Experto piensa y dice: "Bien, ahora abre el archivo auth.js y cambia la línea 5". Este proceso continúa hasta solucionar el bug.
